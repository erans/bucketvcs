# M13 Phase 1 — Batch Handler

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Stand up the LFS Batch API endpoint at `POST /<tenant>/<repo>.git/info/lfs/objects/batch`. After this phase, a stock `git-lfs` client can hit the batch endpoint and receive a structurally correct `BatchResponse` (with placeholder URLs on localfs and real signed URLs on cloud backends thanks to P0). Phase 2 swaps the localfs placeholder for proxied HMAC URLs; Phase 3 adds verify.

**Architecture:** New files `internal/lfs/{batch,handler,metrics,audit}.go` plus targeted edits to `internal/gateway/{routes,server}.go` and `cmd/bucketvcs/serve.go`. `Build` is pure logic — takes the parsed request and a `*Store`, calls `Store.Head`/`PresignPut`/`PresignGet` only. `NewHTTPHandler` wires Build into the HTTP boundary: auth (read-required by `ParseRoute`, write-upgraded inline for upload operation), body parsing, response writing, metric/audit emission. Routes integrate with the existing `routeRepo` dispatch via a new `OpLFSBatch` constant.

**Tech Stack:** Go. Reuses `internal/lfs/{types,errors,store}.go` (P0), `internal/auth` (M4 actor+perm model + `auth.Decide`), `internal/gateway` (ParseRoute / RunAuth / routeRepo), slog-based metric+audit emission matching `internal/gateway/log.go` and `internal/maintenance/log.go`.

## Phase 1 design decisions

These are committed up-front to remove ambiguity from the per-task instructions:

1. **`Build` is pure.** It takes `(ctx, BatchRequest, *Store, verifyBaseURL string, bearerForVerify string, presignTTL time.Duration)` and returns `(BatchResponse, error)`. It calls `Store.Head` on every object, then `Store.PresignPut` or `Store.PresignGet` per the operation. On `ErrNotSupported` from presign, it falls back to `Store.ProxiedPutURL` / `Store.ProxiedGetURL` (which are P0 stubs returning `("", nil)`). The fallback wiring is now established; P2 makes the proxied stubs return real URLs.
2. **Per-object errors vs top-level errors.** Top-level 4xx/5xx only for request-level failures (auth, malformed JSON, missing repo). Per-object errors live inside the `BatchResponse.Objects[i].Error` field, with the top-level response returning HTTP 200. This matches the LFS spec.
3. **Write-permission upgrade lives in the handler, not `ParseRoute`.** `ParseRoute` is body-free, so it cannot know the operation. It returns `RequiredAction = auth.ActionRead`. After the body is parsed, the handler runs a second `auth.Decide` with `ActionWrite` for `operation == "upload"`. Anonymous + public-read repos can drive the read path; only authed actors with `PermWrite` can drive the upload path.
4. **Verify URL is the inbound request's scheme + host.** The handler builds `verifyBaseURL` from `r` as `<scheme>://<r.Host>/<tenant>/<repo>.git/info/lfs/objects` — the per-object verify Href is `<verifyBaseURL>/<oid>/verify`. Phase 3 mounts the verify route at that path.
5. **Verify-action bearer.** The handler captures the inbound `Authorization` header value (Basic + Bearer alike — the LFS verify action just echoes whatever the client used) and passes it to `Build` as the `bearerForVerify` argument. `Build` puts it into the per-object `verify.Header["Authorization"]`. (Phase 4 will refine this for the SSH-issued bearer case.)
6. **Phase 1 returns placeholder URLs on localfs.** The localfs Store path returns `("", nil)` from `ProxiedPutURL/ProxiedGetURL`. `Build` does NOT error on empty URLs — it returns the empty-href action and lets Phase 2 fix it. Tests for localfs LFS assert response shape, not URL validity.
7. **CLI flags default ON.** `--lfs` defaults to `true`, `--lfs-presign-ttl` to `15m`. Spec §8.

```go
package lfs

import (
    "context"
    "errors"
    "fmt"
    "time"

    "github.com/bucketvcs/bucketvcs/internal/storage"
)

// Build is the pure LFS Batch protocol logic: given a parsed request
// and a Store scoped to the target repo, it returns the response shape
// the HTTP handler should serialize. Build performs:
//
//   - Validation of operation (must be "upload" or "download") and
//     transfers (must include "basic" or be empty, which the LFS spec
//     treats as implicit basic).
//   - Per-object Head against the store.
//   - For upload: missing -> upload+verify actions; present with matching
//     size -> empty actions (already uploaded); present with mismatched
//     size -> per-object 422 error.
//   - For download: present -> download action; missing -> per-object 404.
//
// Build never returns top-level errors for per-object conditions. The
// only top-level errors are: unsupported operation (caller -> 422) and
// missing basic transfer (caller -> 422). The handler is responsible
// for mapping those to HTTP responses.
//
// On a presign error other than ErrNotSupported, Build records a
// per-object 500 error rather than poisoning the whole response. On
// ErrNotSupported, Build falls back to Store.ProxiedPutURL /
// ProxiedGetURL (today these return empty strings; Phase 2 wires the
// real proxied HMAC URL).
func Build(ctx context.Context, req BatchRequest, store *Store, verifyBaseURL, bearerForVerify string, presignTTL time.Duration) (BatchResponse, error) {
    if req.Operation != "upload" && req.Operation != "download" {
        return BatchResponse{}, fmt.Errorf("lfs: unsupported operation %q", req.Operation)
    }
    if len(req.Transfers) > 0 {
        // Per the spec, an explicit Transfers list must contain "basic".
        ok := false
        for _, t := range req.Transfers {
            if t == "basic" {
                ok = true
                break
            }
        }
        if !ok {
            return BatchResponse{}, fmt.Errorf("lfs: client did not advertise the 'basic' transfer")
        }
    }
    resp := BatchResponse{Transfer: "basic", Objects: make([]ObjectAction, 0, len(req.Objects))}
    for _, ref := range req.Objects {
        resp.Objects = append(resp.Objects, buildOne(ctx, ref, req.Operation, store, verifyBaseURL, bearerForVerify, presignTTL))
    }
    return resp, nil
}

func buildOne(ctx context.Context, ref ObjectRef, op string, store *Store, verifyBaseURL, bearer string, ttl time.Duration) ObjectAction {
    out := ObjectAction{OID: ref.OID, Size: ref.Size}
    size, exists, err := store.Head(ctx, ref.OID)
    if err != nil {
        out.Error = &ObjectError{Code: 500, Message: "head failed: " + err.Error()}
        return out
    }
    switch op {
    case "upload":
        if exists {
            if size == ref.Size {
                // Already present at the claimed size: empty actions
                // tells git-lfs "skip this upload".
                return out
            }
            out.Error = &ObjectError{Code: 422, Message: fmt.Sprintf("size mismatch: stored=%d, claimed=%d", size, ref.Size)}
            return out
        }
        url, hdr, err := store.PresignPut(ctx, ref.OID, ref.Size, ttl)
        if errors.Is(err, storage.ErrNotSupported) {
            url2, hdr2 := store.ProxiedPutURL(ref.OID, ref.Size, ttl)
            url = url2
            hdr = hdr2
            err = nil
        }
        if err != nil {
            out.Error = &ObjectError{Code: 500, Message: "presign failed: " + err.Error()}
            return out
        }
        out.Actions = map[string]Action{
            "upload": {Href: url, Header: headerMap(hdr)},
            "verify": {
                Href:   verifyBaseURL + "/" + ref.OID + "/verify",
                Header: map[string]string{"Authorization": bearer},
            },
        }
    case "download":
        if !exists {
            out.Error = &ObjectError{Code: 404, Message: "object not found"}
            return out
        }
        url, hdr, err := store.PresignGet(ctx, ref.OID, ttl)
        if errors.Is(err, storage.ErrNotSupported) {
            url2, hdr2 := store.ProxiedGetURL(ref.OID, ttl)
            url = url2
            hdr = hdr2
            err = nil
        }
        if err != nil {
            out.Error = &ObjectError{Code: 500, Message: "presign failed: " + err.Error()}
            return out
        }
        out.Actions = map[string]Action{
            "download": {Href: url, Header: headerMap(hdr)},
        }
    }
    return out
}

// headerMap converts an http.Header (which is map[string][]string) into
// the single-valued map[string]string the LFS wire format uses. For our
// own minted headers each key has at most one value, so we take the
// first element. Nil header -> nil map (matches LFS spec omitempty).
func headerMap(h interface{ Get(string) string }) map[string]string {
    // We accept the *http.Header signature loosely via an interface
    // because the Store's Presign* methods return http.Header and our
    // Proxied* stubs return nil. The interface dodge keeps imports
    // minimal here.
    if h == nil {
        return nil
    }
    // Only the headers we mint matter; today that's just Content-Type.
    if ct := h.Get("Content-Type"); ct != "" {
        return map[string]string{"Content-Type": ct}
    }
    return nil
}
```

```go
package lfs

import (
    "context"
    "log/slog"
)

// emitMetric logs a structured metric in the same shape used by
// internal/gateway/log.go and internal/maintenance/log.go: an info-level
// "metric" record with attributes metric_name (string), value (int64),
// plus key/value pairs from kvs.
func emitMetric(ctx context.Context, logger *slog.Logger, name string, value int64, kvs ...any) {
    if logger == nil {
        logger = slog.Default()
    }
    attrs := []slog.Attr{
        slog.String("metric_name", name),
        slog.Int64("value", value),
    }
    for i := 0; i+1 < len(kvs); i += 2 {
        k, ok := kvs[i].(string)
        if !ok {
            logger.LogAttrs(ctx, slog.LevelDebug, "emitMetric: skipping non-string label key",
                slog.String("metric_name", name),
                slog.Any("bad_key", kvs[i]))
            continue
        }
        attrs = append(attrs, slog.Any(k, kvs[i+1]))
    }
    logger.LogAttrs(ctx, slog.LevelInfo, "metric", attrs...)
}

// emitBatchRequestMetric increments lfs_batch_requests_total{op,result}.
// op is "upload" or "download". result is one of:
//
//   - "ok": batch returned 200 with at least one object processed.
//   - "unauthorized": 401.
//   - "forbidden": 403.
//   - "notfound": 404 (repo not found).
//   - "error": any other 4xx or 5xx, including 422 on malformed body.
func emitBatchRequestMetric(ctx context.Context, logger *slog.Logger, op, result string) {
    emitMetric(ctx, logger, "lfs_batch_requests_total", 1,
        "op", op,
        "result", result,
    )
}

// emitBatchObjectMetric increments lfs_batch_objects_total{op,result}.
// Called once per object in a batch response. result is one of:
//
//   - "new": upload that produced an upload action (object missing).
//   - "exists": object present at matching size — upload returned empty
//     actions OR download returned a download action.
//   - "missing": download for an object not present.
//   - "error": any per-object error (size mismatch, presign failure,
//     etc.).
func emitBatchObjectMetric(ctx context.Context, logger *slog.Logger, op, result string) {
    emitMetric(ctx, logger, "lfs_batch_objects_total", 1,
        "op", op,
        "result", result,
    )
}
```

```go
package lfs

import (
    "context"
    "encoding/json"
    "errors"
    "log/slog"
    "net/http"
    "path"
    "strings"
    "time"

    "github.com/bucketvcs/bucketvcs/internal/auth"
)

// Deps is the dependency surface NewHTTPHandler needs. It is kept as
// a value-type struct so callers (the gateway today, a future test
// harness) configure exactly what they need without touching package
// internals.
type Deps struct {
    // AuthStore is consulted both for repo existence (404 path) and
    // for credential + permission resolution.
    AuthStore auth.Store

    // NewStore returns the per-repo lfs.Store for the given (tenant,
    // repo). The gateway constructs this by combining its top-level
    // storage.ObjectStore with the repo's prefix.
    NewStore func(tenant, repo string) *Store

    // PresignTTL is the TTL passed into Store.PresignPut/PresignGet.
    PresignTTL time.Duration

    // Logger is used for metric + audit emission. Nil falls back to
    // slog.Default() (same shape as internal/gateway/log.go).
    Logger *slog.Logger
}

// NewHTTPHandler returns the http.Handler that serves the LFS Batch
// endpoint. The handler authenticates via the same scheme used by the
// Git smart-HTTP path (M4 basic auth via auth.Store), upgrades to a
// secondary write check when the body's operation is "upload", and
// emits the lfs.batch audit event plus lfs_batch_requests_total /
// lfs_batch_objects_total metrics.
//
// The handler is intentionally constructed with a closure over deps so
// the gateway can mount it via mux.Handle without further wiring.
func NewHTTPHandler(deps Deps) http.Handler {
    return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
        ctx := r.Context()
        logger := deps.Logger
        if logger == nil {
            logger = slog.Default()
        }

        // Parse path: /<tenant>/<repo>.git/info/lfs/objects/batch
        tenant, repo, rest, ok := splitLFSPath(r.URL.Path)
        if !ok || rest != "info/lfs/objects/batch" || r.Method != http.MethodPost {
            http.NotFound(w, r)
            return
        }

        // 404 on missing repo (before any auth challenge — matches Git
        // smart-HTTP behavior in gateway/auth.go).
        flags, err := deps.AuthStore.GetRepoFlags(ctx, tenant, repo)
        if errors.Is(err, auth.ErrNoSuchRepo) {
            WriteError(w, http.StatusNotFound, "repository not found")
            emitLFSBatch(ctx, logger, tenant+"/"+repo, "", "", 0, "notfound")
            emitBatchRequestMetric(ctx, logger, "", "notfound")
            return
        }
        if err != nil {
            WriteError(w, http.StatusInternalServerError, "internal error")
            emitBatchRequestMetric(ctx, logger, "", "error")
            return
        }

        // Authn via Basic auth (mirrors gateway.RunAuth behavior, but
        // returns LFS-shaped errors rather than Git-shaped 401).
        var actor *auth.Actor
        if user, pass, hasBasic := r.BasicAuth(); hasBasic {
            a, _, _, verr := deps.AuthStore.VerifyCredential(ctx, auth.BasicPassword{Username: user, Password: pass})
            if verr != nil {
                w.Header().Set("WWW-Authenticate", `Basic realm="bucketvcs"`)
                WriteError(w, http.StatusUnauthorized, "invalid credentials")
                emitBatchRequestMetric(ctx, logger, "", "unauthorized")
                return
            }
            actor = a
        }

        // Parse body BEFORE the write-permission check, so we know op.
        body := http.MaxBytesReader(w, r.Body, 1<<20) // 1 MiB hard cap on Batch body
        var req BatchRequest
        if err := json.NewDecoder(body).Decode(&req); err != nil {
            WriteError(w, http.StatusUnprocessableEntity, "unprocessable: "+err.Error())
            emitBatchRequestMetric(ctx, logger, "", "error")
            return
        }
        if req.Operation != "upload" && req.Operation != "download" {
            WriteError(w, http.StatusUnprocessableEntity, "unsupported operation")
            emitBatchRequestMetric(ctx, logger, req.Operation, "error")
            return
        }

        // Permission decision. Read is the default; upload upgrades to
        // write.
        var perm auth.Perm
        if actor != nil {
            p, perr := deps.AuthStore.LookupRepoPerm(ctx, actor, tenant, repo)
            if perr != nil {
                WriteError(w, http.StatusInternalServerError, "internal error")
                emitBatchRequestMetric(ctx, logger, req.Operation, "error")
                return
            }
            perm = p
        }
        needAction := auth.ActionRead
        if req.Operation == "upload" {
            needAction = auth.ActionWrite
        }
        if ok, _ := auth.Decide(actor, perm, needAction, flags); !ok {
            if actor == nil {
                w.Header().Set("WWW-Authenticate", `Basic realm="bucketvcs"`)
                WriteError(w, http.StatusUnauthorized, "unauthorized")
                emitBatchRequestMetric(ctx, logger, req.Operation, "unauthorized")
            } else {
                WriteError(w, http.StatusForbidden, "forbidden")
                emitBatchRequestMetric(ctx, logger, req.Operation, "forbidden")
            }
            return
        }

        // Build the response.
        store := deps.NewStore(tenant, repo)
        verifyBaseURL := requestBaseURL(r) + "/" + tenant + "/" + repo + ".git/info/lfs/objects"
        bearerForVerify := r.Header.Get("Authorization")
        resp, berr := Build(ctx, req, store, verifyBaseURL, bearerForVerify, deps.PresignTTL)
        if berr != nil {
            WriteError(w, http.StatusUnprocessableEntity, berr.Error())
            emitBatchRequestMetric(ctx, logger, req.Operation, "error")
            return
        }

        // Emit per-object metrics.
        for _, o := range resp.Objects {
            emitBatchObjectMetric(ctx, logger, req.Operation, perObjectResult(req.Operation, o))
        }

        // Write 200.
        w.Header().Set("Content-Type", ContentType)
        w.WriteHeader(http.StatusOK)
        _ = json.NewEncoder(w).Encode(resp)

        // Audit + request-level metric.
        emitBatchRequestMetric(ctx, logger, req.Operation, "ok")
        user := ""
        if actor != nil {
            user = actor.Username
        }
        emitLFSBatch(ctx, logger, tenant+"/"+repo, user, req.Operation, len(resp.Objects), "ok")
    })
}

// splitLFSPath splits "/<tenant>/<repo>.git/<rest>" into its parts.
// Returns ok=false on any structural mismatch.
func splitLFSPath(p string) (tenant, repo, rest string, ok bool) {
    if p != path.Clean(p) {
        return "", "", "", false
    }
    parts := strings.SplitN(strings.TrimPrefix(p, "/"), "/", 3)
    if len(parts) < 3 {
        return "", "", "", false
    }
    tenant = parts[0]
    repoSeg := parts[1]
    rest = parts[2]
    if !strings.HasSuffix(repoSeg, ".git") || repoSeg == ".git" {
        return "", "", "", false
    }
    repo = strings.TrimSuffix(repoSeg, ".git")
    if tenant == "" || repo == "" {
        return "", "", "", false
    }
    return tenant, repo, rest, true
}

// requestBaseURL reconstructs "<scheme>://<host>" from the inbound
// request. For requests reaching us behind a TLS-terminating proxy,
// callers SHOULD set X-Forwarded-Proto and X-Forwarded-Host; we honor
// X-Forwarded-Proto when set.
func requestBaseURL(r *http.Request) string {
    scheme := "http"
    if r.TLS != nil {
        scheme = "https"
    }
    if xfp := r.Header.Get("X-Forwarded-Proto"); xfp != "" {
        scheme = xfp
    }
    return scheme + "://" + r.Host
}

// perObjectResult returns the metric label for one ObjectAction. The
// label vocabulary matches the spec §7: new|exists|missing|error.
//
//   - upload, missing -> "new" (upload action returned).
//   - upload, present at matching size -> "exists" (empty actions).
//   - upload, present at mismatched size -> "error" (per-object 422).
//   - download, present -> "exists" (download action returned).
//   - download, missing -> "missing" (per-object 404).
//   - any other per-object error (presign failure, head error) -> "error".
func perObjectResult(op string, o ObjectAction) string {
    if o.Error != nil {
        if op == "download" && o.Error.Code == 404 {
            return "missing"
        }
        return "error"
    }
    if op == "upload" && len(o.Actions) == 0 {
        return "exists"
    }
    if op == "download" {
        return "exists"
    }
    return "new"
}
```
